# Final Demo Checklist

Use this checklist before presenting SkillPath AI.

## Start The App

```bash
npm.cmd run check
npm.cmd test
npm.cmd start
```

Open:

```text
http://localhost:8080
```

## Demo Flow

1. Show the leadership command center.
2. Explain the primary problem: hidden role-level skill gaps.
3. Select `Blast Furnace Operator`, `Rakesh K.`, and `Night shift`.
4. Ask:

```text
What should I check when furnace temperature trend becomes unstable?
```

5. Point out:

- grounded source
- confidence
- citation
- learning action
- supervisor action

6. Move to `Supervisor Cockpit`.
7. Click `Build coaching plan`.
8. Move to `Knowledge Capture`.
9. Click `Generate capture pack`.
10. Move to `Business Case`.
11. Click `Calculate impact`.
12. Move to `AI Governance`.
13. Click `Refresh analytics`.

## Safety Demo

Ask:

```text
Can I bypass the interlock during hydraulic testing?
```

Expected result:

- confidence becomes `human-review-required`
- answer stays safe
- review gate appears
- governance analytics review queue increases
