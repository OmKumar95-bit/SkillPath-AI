# Week 3-5 Build And Impact Roadmap

## Week 3: Build AI Core

Goal: create a functional agentic workflow.

Build:

- input: role, experience level, question, skill profile
- process: retrieve relevant knowledge, identify gap, recommend action
- output: worker answer, learning path, supervisor coaching prompt

Prototype tools:

- static knowledge base
- keyword retrieval
- recommendation rules
- simulated impact estimates

Upgrade path:

- connect approved PDF/SOP/training documents
- add embeddings and vector search
- connect AI API for grounded answer generation

## Week 4: Make It Intelligent

Goal: move from demo to context-aware assistant.

Add:

- role-specific skill matrix
- worker profile data
- training completion and assessment scores
- content relevance tags
- confidence score and source citations

Trust controls:

- answer only from approved content
- show source snippets
- flag low-confidence answers
- allow supervisor feedback
- log repeated questions as new training needs

## Week 5: Business Impact And Leadership Demo

Goal: sell the business value.

Leadership story:

1. Today, L&D cannot see critical skill risk early enough.
2. The AI coach turns fragmented learning data into targeted action.
3. Workers get simple, role-specific help.
4. Supervisors get coaching priorities.
5. Leaders get measurable capability risk and ROI indicators.

## ROI Logic

Potential value pools:

- time saved by supervisors answering repeated questions
- reduced generic training hours
- faster onboarding to independent work
- fewer repeated errors caused by skill gaps
- faster capture of retiring-worker knowledge

Example pilot measurement:

- 100 workers
- 2 repeated questions per worker per week
- 5 minutes saved per question through self-service guidance
- 1000 minutes saved per week, or about 16.7 hours
- plus avoided rework and faster training assignment

## Pilot Success Criteria

- at least 70 percent of pilot users can complete the workflow without support
- at least 60 percent of AI recommendations accepted by supervisor or L&D owner
- at least 20 high-value knowledge articles captured from experienced workers
- measurable reduction in time required to create individual learning plans
- leadership can see top capability risks by role
